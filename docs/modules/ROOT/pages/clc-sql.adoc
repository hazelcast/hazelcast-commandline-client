= clc sql
:description: Executes a SQL statement.

{description}

Usage:

[source,bash]
----
hzc sql "[query]" [--format OUTPUT-FORMAT]
----

Parameters:

[cols="1m,1a,2a,1a"]
|===
|Parameter|Required|Description|Default

|`--format`, `-f`
|Optional
|Output format. Supported formats:

- `csv`
- `delimited`
- `json`
- `table`
|`delimited` in non-interactive mode, `table` in interactive mode.

|===

== Querying with SQL using Hazelcast CLC

You can run SQL queries in both interactive and non-interactive mode of Hazelcast CLC.

[tabs]
====
Non-Interactive mode::
+
--
[source,bash]
----
clc sql "SELECT * FROM employees"
+-----------------------------------------------------------------+
|        __key        |         age         |         name        |
+-----------------------------------------------------------------+
| 12                  | 41                  | Jane Brown          |
| 5                   | 33                  | Mandy Bronson       |
| 21                  | 30                  | Mike McGregor       |
| 3                   | 22                  | Joe Taylor          |
| 75                  | 33                  | Mandy Bronson       |
----
--
Interactive mode::
+
--
[source,bash]
----
clc
CLC> sql "SELECT * FROM employees"
+-----------------------------------------------------------------+
|        __key        |         age         |         name        |
+-----------------------------------------------------------------+
| 12                  | 41                  | Jane Brown          |
| 5                   | 33                  | Mandy Bronson       |
| 21                  | 30                  | Mike McGregor       |
| 3                   | 22                  | Joe Taylor          |
| 75                  | 33                  | Mandy Bronson       |
----
--
====
